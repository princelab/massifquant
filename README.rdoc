= kalmanquant

Uses Kalman tracking to quantify chromatographic peaks in a bin-less manner.
Operates on previously centroided data. 

This is a rewrite of an original implementation by Christopher Conley of a
work by Aberg et al. ({J Chromatogr A. 2008 May
23;1192(1):139-46}[http://www.ncbi.nlm.nih.gov/pubmed/18378252]).
